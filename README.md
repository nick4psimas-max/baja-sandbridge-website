# Baja Sandbridge Website Redesign
Website redesign project for Baja Sandbridge (Wix + AI tools)


## Overview
This project is a full redesign of the Baja Sandbridge website focused on improving user experience, visual quality, and lead conversion.

## Impact
Improved visual quality and site flow to create a more premium, conversion-focused user experience.

## What I Did
- Rebuilt site structure using Wix and AI tools
- Optimized layout for mobile and desktop responsiveness
- Designed a clean, high-end visual experience aligned with a water front resturant
- Improved navigation and call-to-action placement for better user flow

## Goal
Create a professional, scalable website that can be easily handed off to the business owner.

- Lovable.dev
- AI design tools
